Ultrasonic Walking Cane



By: Nolan Santellanes & Miguel Almejo


Overview: This project was one of our final projects for the semester and its functions was to use a potentiometer to move the ship and an onboard push button to shoot along with sonund hear the shots and enemies being destroyed
The components used where:

  • TM4C123 Microcontroller
  • Nokia 5110 LCD Display
  • 10K ohm potentiometer
  • Speaker
  • LM386 Amplifier
  • 4 x 2K ohm resistors
  • 4 x 4K ohm resistors
Description: The game start with a prompt that asks the user to press SW2 on the board to start the game. Once pressed, the game starts with 3 invader and the user has shoot down the invaders using SW1 on the board and also move with the potentiometer to win the game. Whether the user managed to shoot down all the enemies or not the end screen shows the number of enemies shot down and the start prompt is shown again after three seconds.
